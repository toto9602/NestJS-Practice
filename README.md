## 레포지토리 관리
0. `master` 브랜치에는 테스트를 위한 기초적인 세팅만 해 두었습니다.
1. 연습해 보고 싶은 기능 혹은 내용이 있다면, 이를 `Issue`로 작성합니다.
2. `Issue`에 대한 브랜치를 생성합니다. <br/>
   ex) `feat/#이슈_번호/기능_설명`
3. 기능 연습 후, 해당 브랜치에서 `master` 브랜치로 PR을 생성합니다.
